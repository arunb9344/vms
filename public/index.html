ONVIF Configuration Steps

To connect successfully, you must configure the following settings inside your camera's admin web portal:

  • Enable ONVIF Protocol:
    Many cameras (Hikvision, Prama, Dahua) disable ONVIF access by default. In the camera's web UI: Go to Network -> Advanced Settings -> Integration Protocol, and check "Enable ONVIF".
  • Add an ONVIF User:
    In the same ONVIF tab, click "Add" to create a dedicated user (e.g. onvif or admin) and set the type to Media/Administrator. Using your camera's primary web admin user password directly will fail unless registered as an ONVIF user here.
  • Confirm the ONVIF Port:
    Typically, Hikvision and Prama use 80 (or 8000/8080/8899). Dahua uses 80 or 37777. CP Plus often uses 8899. Check your camera's network port settings page.
  • Test local reachability:
    Ensure your PC is on the same local subnet as the camera IP (e.g., both on `192.168.0.x`).
